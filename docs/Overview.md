# Generic API Client

# Purpose

I created this package because I wanted a simple and standard way to interact with API for a developer.
The goal of the package is to separate the data access layer, the requests templates , from the business logic, the python methods.
This provide a way to abstract how the API works so the developer can focus on something else.

# Installation

Just run the following command to install the package from Pypi

```bash
pip install generic-api-client
```

# Usage

The usage of the tool is easy but it requires a bit of code to declare how the API you want to connect with works.

The best way to startup is to follow this [tutorial](Nextcloud-Example)
